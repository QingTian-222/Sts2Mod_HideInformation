# 信息隐藏mod

[发布页下载](https://github.com/QingTian-222/Sts2Mod_HideInformation/releases/tag/v1.0)

## 安装方法

1. 下载发布页的压缩文件。
2. 将整个解压缩后的模组文件夹放入游戏根目录的 `mods` 文件夹中。
3. 可以更改HideInformation里的config.txt，将对应条目改为true/false来开启或关闭效果
4. 兼容皮皮配置Modconfig模组，安装皮皮配置后可以在游戏内热切换

## 版本适配

本模组无依赖，支持几乎任意版本

## mod详情

- 可选隐藏遗物图标替换为统一可自定义图标
- 可选隐藏文本替换为统一可自定义文本
- 可选隐藏生物贴图场景
- 可选隐藏地图标志统一为问号
- 可选隐藏怪物意图
- 可选隐藏卡图替换为统一可自定义卡图
